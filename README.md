## 기능 목록

### 사다리 게임

- [x] 참가자들의 수, 경품 수가 같아야한다.
- [x] 특정 참가자의 결과를 알 수 있다.

### 사다리

- [x] 높이가 양수가 아니면 IllegalArgumentException을 던진다.
- [x] 너비는 일정하지 않으면 IllegalArgumentException을 던진다.
- [x] 시작 위치로 목적지 위치를 알 수 있다.

### 가로줄

- [x] 연속되는 다리가 있으면 IllgalArgumentException을 던진다.
- [x] 현재 위치에서 이동할 다음 방향을 알 수 있다.
  - 좌측, 유지, 우측

### 다리

- [x] 주어지는 다리와 연속인지 알 수 있다.
- [x] 다리가 있는지 알 수 있다.

### 다리 생성기

- [x] 다리 하나를 생성한다

### 가로줄 생성기

- [x] 라인 하나를 생성한다
- [x] 한 라인은 최대 (사람 수 - 1)개의 브릿지를 가진다
- [x] 브릿지가 연속되지 않게 생성한다.

### 위치

- [x] 현재 위치의 왼쪽, 오른쪽을 알 수 있다.
- [x] 특정 범위 사이에 있는지 알 수 있다.
- [x] 특정 방향으로 움직일 수 있다.

### 상품

- [x] 특정 위치의 상품을 알 수 있다
- [x] 특정 크기를 가지는지 알 수 있다

### 참가자

- [x] 이름은 최소 1글자 이상 최대 5글자 이하여야한다.
- [x] 참가자 이름이 같으면 같은 참가자로 취급한다

### 참가자들

- [x] 2명 이상이어야 한다
- [x] 중복될 수 없다
- [x] 특정 참가자의 시작점을 알 수 있다.
    - [x] 특정 참가자를 찾지 못한 경우 예외를 던진다.

### 사다리 게임 엔진

- [x] 참가자를 모집한다.
- [x] 경품을 입력받는다.
- [x] 사다리 높이를 입력받는다.
- [x] 사다리를 만든다.
- [x] 만든 사다리를 출력한다.
- [x] 결과를 보고싶은 사람의 경품을 출력한다.
    - [x] "all"은 모든 이의 경품을 출력하고 게임을 종료한다.

### 입력

- [x] 사람 이름은 "," 로 구분한다.
  - [x] 명령어는 사람 이름으로 사용할 수 없다.
- [x] 사다리 높이를 입력받는다.
- [x] 실행 결과(경품)들을 입력받는다.
- [x] 결과를 보고싶은 사람을 입력받는다.

### 출력

- [x] 사다리를 출력한다.
    - [x] 참가자 이름을 출력한다.
    - [x] 브릿지들을 출력한다.
    - [x] 경품들을 출력한다.
- [x] 결과를 출력한다.

### 예외 처리

- [x] 잘못된 입력부터 다시 받는다
